## About The Project

**Suichat Server** is a conceptual extension of a university Systems Programming lab to create a multithreaded chat server. This application builds upon the idea by incorporating physical portability and deployment onto a microcontroller, granting me experience with: the PlatformIO toolchain, FreeRTOS, and Arduino's WiFi module.
